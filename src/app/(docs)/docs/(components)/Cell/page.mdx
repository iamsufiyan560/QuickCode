## Props

### Cell

| Prop        | Type                                | Default     | Description            |
| ----------- | ----------------------------------- | ----------- | ---------------------- |
| `variant`   | `"default" \| "outline" \| "muted"` | `"default"` | Visual style variant   |
| `size`      | `"default" \| "sm"`                 | `"default"` | Size of the cell       |
| `className` | `string`                            | —           | Additional CSS classes |
